# vite-plugin-assetpack

Using assetpack in Vite

## Usage
### install
```
npm install vite-plugin-assetpack -D
// yarn add vite-plugin-assetpack -D

```
### vite.config.[ts|js|...]

```
import  {viteAssetPackPlugin} from "vite-plugin-assetpack"

export default defineConfig({
    ...
    plugins: [
    viteAssetPackPlugin(),
    ],
})
```
